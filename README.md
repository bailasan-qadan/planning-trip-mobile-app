## Project Overview
This project is an Android mobile application developed using **Android Studio**.  
The application helps users plan trips by organizing destinations, schedules, and trip details through a simple and user-friendly interface.

The project focuses on applying mobile application development concepts, UI design, and screen navigation using Android technologies.

---

## Features
- Plan and organize trips
- Add destinations and trip details
- Simple and clean user interface
- Easy navigation between application screens

---

## Technologies Used
- Android Studio
- Java / Kotlin
- XML (UI Design)
- Android SDK

---

## Project Structure
The project follows the standard Android Studio structure:
- Activities handle the application logic and navigation
- XML layout files define the user interface
- Resources include colors, styles, and string values
